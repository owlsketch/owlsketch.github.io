Photocodegraphy

  • Video
    • |
  • Download
    • |
  • Source Code

Photocodegraphy is a mixture of photography with code. It is a new medium that is the result of manipulating imagery with logic based graphics, implemented through a variety of APIs mainly canvas and webGL. It is a motion picture, but is not pre-rendered, that is, the animation seen in these images are dynamically generated when observed, and have random processes applied so that no two viewers technically see the same composition.

This was my first visual project after hearing about creative coding through PBS on youtube. I used openFrameworks and C++ and made this in a few hours. It is an audio visualizer program that moves to the beat of the music it is playing. Made in January, 2014, again with good intentions. I plan on re-implementing this in webGL in order to allow portability and remove the OS specific restriction.
